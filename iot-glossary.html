IoT Glossary

Essential Terms and Definitions in Internet of Things Technology

Communication Protocols

Standard protocols and communication technologies

Wireless Protocols
  • MQTT (Message Queuing Telemetry Transport) Lightweight messaging protocol for small sensors and mobile devices
  • LoRaWAN (Long Range Wide Area Network) Low-power protocol for long-range IoT communications
  • Zigbee Low-power, short-range wireless communications protocol
Network Protocols
Internet Protocols
  • CoAP (Constrained Application Protocol) Internet protocol for constrained devices
  • HTTP/HTTPS Standard web protocols for IoT data transmission
  • WebSocket Protocol for full-duplex communication channels
Web Protocols

Hardware Components

Essential IoT hardware terminology

Processing Units
  • MCU (Microcontroller Unit) Integrated circuit containing processor, memory, and I/O peripherals
  • SBC (Single Board Computer) Complete computer built on a single circuit board
  • SoC (System on Chip) Integrated circuit integrating all components of a computer
Hardware Components
Sensors & Actuators
  • Sensor Node Device that contains sensors, processor, and communication module
  • Actuator Device that converts energy into mechanical motion
  • Gateway Device that connects IoT devices to the broader network
Input/Output Devices

Software & Services

Cloud and software terminology

Cloud Services
  • IoT Platform Cloud-based service for managing IoT devices and data
  • Edge Computing Data processing at or near the source of data generation
  • Cloud Computing Computing services delivered over the internet
Cloud Technologies
Data Management
  • Data Analytics Process of examining data sets to draw conclusions
  • Time Series Database Database optimized for time-stamped data
  • Data Lake Storage repository that holds raw data
Data Systems

Note on Technical Terms

IoT terminology evolves rapidly with technological advancement. Stay updated with the latest terms and definitions through industry publications and technical documentation.
